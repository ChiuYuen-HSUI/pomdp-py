pomdp_py.representations.belief.histogram.update_histogram_belief(current_histogram, real_action, real_observation, observation_model, transition_model, oargs={}, targs={}, normalize=True)¶

This update is based on the equation: \(B_{new}(s') = n O(z|s',a) \sum_s T(s'|s,a)B(s)\).

current_histogram is the Histogram that represents current belief.

Returns the histogram distribution as a result of the update
